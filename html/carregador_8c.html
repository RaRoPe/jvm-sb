Arquivo que contém funções para carregar classes nas estruturas adequadas.

Funções

classFile* buscaClasseIndice ( int  indice)

Função que retorna uma referencia a uma classFile de acordo com o indice.
Util para encontrar a classe da main no inicio do processamento.

Parâmetros
Posicaoa ser acessada no array de classes.
Retorna
Referencia a um classFile.
int32_t carregaMemClasse ( char *  nomeClass)

É passado o nome do arquivo .class a ser carregado.
Se ja foi carregado retorna a posição no array de classes da classe. Se ainda não foi carregado carrega no array de classes.

Parâmetros
nomedo arquivo ".class" a ser carregado no array de classes.
Retorna
posição da classe alocada no array de classes.
char* retornaNomeClasse ( classFile *  classe)

Acessa a constant pool no campo bytes a partir do index da classe e retorna o nome da classe.
Necessaria para saber se classe ja foi carregada ou não.

Parâmetros
EstruturaclassFile de que se deseja obter o nome.
Retorna
nome da classe fornecida.

Variáveis

classFile** arrayClasses

Array de referencia a classFiles. Declarado no carregador.h

int16_t numeroClasses = 0

Quantidade de classes já carregadas pelo carregador.
