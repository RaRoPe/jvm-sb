Ponteiro para uma estrutura de \hyperlink{structmethod__info}{method\-\_\-info}. Para comecar executando a main.

1 -\/ Inicializa as instrucoes implementadas pela jvm.

2 -\/ Verifica argumentos por linha de comando.

3 -\/ Carrega no array de classes o .class passado por linha de comando.

4 -\/ Busca o metodo main para comecar a execucao.

5 -\/ Cria frame e coloca na pilha. Passa o metodo com o bytecode e a classe que contem o metodo com a constant\-Pool.

6 -\/ Executa o metodo main que esta no topo da \hyperlink{structstackFrame}{stack\-Frame}.
